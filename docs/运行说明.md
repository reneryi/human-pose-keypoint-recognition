# 运行说明

## 一、环境准备

本项目使用 conda 环境 `jc_env`，Python 版本为 3.10 及以上。

检查环境：

```powershell
& "C:\Users\reneryi\Miniconda3\condabin\conda.bat" run -n jc_env python --version
```

安装依赖：

```powershell
& "C:\Users\reneryi\Miniconda3\condabin\conda.bat" run -n jc_env pip install -r requirements.txt
```

## 二、启动程序

在项目根目录执行：

```powershell
& "C:\Users\reneryi\Miniconda3\condabin\conda.bat" run -n jc_env streamlit run app.py
```

启动后浏览器会打开本地页面。如果没有自动打开，可以根据终端输出访问本地地址。

## 三、v1.0.0 图片人体关键点识别

1. 打开程序页面。
2. 上传一张包含人体的图片。
3. 点击“开始识别图片关键点”。
4. 查看原图与识别结果图。
5. 结果会保存到 `outputs/image_result/` 目录，并可通过页面按钮下载。

## 四、命令行测试图片识别

也可以直接调用图片识别模块：

```powershell
& "C:\Users\reneryi\Miniconda3\condabin\conda.bat" run -n jc_env python pose_image.py assets/sample_image.jpg
```

说明：如果没有准备 `assets/sample_image.jpg`，请替换为本机实际图片路径。

## 五、v1.0.1 视频人体关键点识别

1. 启动 Streamlit 程序。
2. 在页面左侧功能选择中切换到“视频关键点识别”。
3. 上传一个视频文件，支持 `mp4`、`avi`、`mov` 和 `mkv` 格式。
4. 点击“开始识别视频关键点”。
5. 程序会逐帧读取视频并绘制人体关键点和骨架。
6. 处理后的视频会保存到 `outputs/video_result/` 目录，并可在页面中预览和下载。

## 六、命令行测试视频识别

也可以直接调用视频识别模块：

```powershell
& "C:\Users\reneryi\Miniconda3\condabin\conda.bat" run -n jc_env python pose_video.py assets/sample_video.mp4 --max-frames 30
```

说明：如果没有准备 `assets/sample_video.mp4`，请替换为本机实际视频路径；`--max-frames` 可用于限制测试帧数，缩短处理时间。
