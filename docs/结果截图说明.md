## 二、Git 操作截图

### 1. 提交历史截图

建议文件名：

```text
screenshots/git_log.png
```

截图前可执行：

```powershell
git log --oneline --decorate --graph -20
```

截图中应能看到多次语义清晰的提交记录，以及 `v1.0.0`、`v1.0.1` 等标签信息。

### 2. 分支截图

建议文件名：

```text
screenshots/branches.png
```

截图前可执行：

```powershell
git branch -a
```

截图中应能看到：

1. `master` 分支。
2. `feature/video-pose` 分支。
3. 远程分支信息。

### 3. 标签截图

建议文件名：

```text
screenshots/tags.png
```


